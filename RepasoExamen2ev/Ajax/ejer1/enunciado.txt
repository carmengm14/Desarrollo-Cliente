Realizar un ejercicio con dos enlaces a dos páginas web para que cada vez que se pinche
en uno de los enlaces se cargue la página correspondiente de forma asíncrona . Las paginas
html deben estar alojadas en el mismo lugar que la principal. No deben tener cabecera HTML,
HEAD, TITLE ya que se incluyen como contenido dentro de otra página web.
En este ejemplo que os visualizo al pinchar sobre inicio sale la página de inicio.html, cuyo código es el siguiente:
<h2 class="aligncenter">INICI</h2>
<p class="aligncenter">
<em>Zona de continguts</em>
</p>
